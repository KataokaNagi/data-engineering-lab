## 実験の概要
- ストップワードの除去
    - Python
- ニュースアイテムからの情報抽出
    - **GATE** (General Architecture For Text Engineering) 
- 文書のアノテーション
    - **Gazetteer**
        - 文書内のエンティティのカテゴリ識別
        - カスタムリスト
        - ベンガル語専用
        - **ANNIE** (A Nearly New Information Extraction System) gazetteer
            - 英語用
    - Python NLTK POS tagger
        - ベンガル語の品詞識別
        - 名前付きエンティティ（NE）
            - 本文の名詞
        - Concept
            - ヘッドラインの名詞
- 形容詞の命名
    - Action
    - Adj
- オントロジー
    - **JAPE** (Jape Annotation Pattern Engine) 
    - Python
- 言語間のマッピング
    - Google 翻訳
    - 対訳シソーラス
    - Python
    - 文書オントロジー間の単語ごと
- **OnAGUI** (Ontology Alignment Graphical User Interface) 
    - ソースオントロジーとターゲットオントロジーの類似性を計算
    - 結果は**RDF**（Resource Description Framework）
